# 📊 Social Media Ads Analytics Dashboard – Power BI

## 📌 Project Overview
This project analyzes **social media advertising campaign performance** using an interactive **Power BI dashboard**. The goal was to transform raw marketing campaign data into actionable insights that help businesses understand engagement patterns, optimize ad strategies, and improve decision-making.

The dashboard analyzes **50+ social media ad campaigns** across different platforms, regions, audiences, and time periods to identify performance drivers and marketing opportunities.

---

## 🎯 Objectives
- Analyze social media advertising performance
- Identify high-performing platforms and regions
- Understand audience behavior by age group and gender
- Evaluate the effectiveness of different ad formats
- Track key marketing KPIs such as CTR and engagement rate

---

## 📈 Key Insights
- **CTR:** 11.79%  
- **Engagement Rate:** 15.91%  
- Certain platforms and countries consistently generated higher engagement.
- Audience engagement varied significantly by **age group and gender**.
- Some **ad formats and creative types** produced stronger engagement results.
- **Monday evenings** showed the highest engagement levels.
- Performance gaps between campaigns became clearly visible through KPI tracking.

---

## 📊 Dashboard Features
- Campaign performance overview with KPI cards
- Platform-wise engagement comparison
- Country and regional performance analysis
- Audience demographic insights
- Ad format effectiveness analysis
- Time-based engagement trend analysis
- Interactive filters and slicers for dynamic exploration

---
